# Gold_Price_Evolution-2000-2024-
project analyzing historical gold prices (2000–2026) using Python, SQL, and Power BI to deliver business-oriented insights, an interactive dashboard, and executive-ready reporting.

Overview

This project presents an end-to-end data analytics workflow applied to historical gold price data.
The objective is to analyze long-term price dynamics, identify market regimes, and translate analytical insights into decision-ready outputs.

The project covers the full analytics lifecycle:

- Data exploration and preparation using Python
- Business-oriented analysis using SQL
- Interactive visualization with Power BI
- Insight communication through a written report and executive presentation

This project is designed as a portfolio-ready case study demonstrating technical skills, analytical reasoning, and business communication.

Dataset

- Subject: Historical gold prices (USD per troy ounce)
- Time range: August 2000 – January 2026
- Frequency: Monthly

Sources:
  - Yahoo Finance (GC=F)
  - World Bank commodity data

Structure:
- Date (time index)
- Gold price (USD)
- Derived calendar and period variables

Tools & Technologies

- Python: Data loading, EDA, cleaning, feature engineering: pandas, numpy
- SQL: Business analysis queries: MySQL Workbench

Dashboard Overview

The Power BI dashboard focuses on:

- Long-term gold price trends
- Regime-based price behavior
- Annual acceleration and stress detection
- High-level KPIs for strategic interpretation

Key Results

- Gold shows strong long-term appreciation with clear regime shifts
- Performance is non-linear, with returns concentrated in stress periods
- Average growth metrics (CAGR) are meaningful but must be complemented by annual variation analysis
- Gold behaves as a regime-sensitive strategic asset, not a short-term trading instrument
